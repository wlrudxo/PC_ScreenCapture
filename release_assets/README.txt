Activity Tracker 사용 가이드
==============================

1. 실행
   - ActivityTracker.exe 실행
   - 시스템 트레이에 아이콘 생성됨

2. Chrome URL 추적 (선택사항)
   - chrome_extension.zip 압축 해제
   - Chrome > 확장 프로그램 > 개발자 모드 ON
   - "압축해제된 확장 프로그램을 로드합니다" 클릭
   - 해제한 폴더 선택

3. 종료
   - 시스템 트레이 아이콘 우클릭 > 종료
   - 설정 탭 하단 > 종료
